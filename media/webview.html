Guide d'utilisation

  • 1. 📂 Charger un Graphe : Cliquez sur le bouton "Charger graph.json".
  • 2. 🔍 Filtres d'Exploration : Le panneau principal permet de cibler votre recherche.
  • 3. 🖱️ Sélection Multiple (Graph) : Maintenez Ctrl / Cmd et cliquez sur un nœud dans le graphe pour l'ajouter ou le retirer de la sélection.
  • 4. 📏 Onglet SÉLECTION (Split Pane) : Glissez la barre verticale entre l'arbre et le graphe pour redimensionner les panneaux.
